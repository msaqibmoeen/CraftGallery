# 🎨 Craft Gallery

Craft Gallery is a web-based platform that showcases handmade crafts, artwork, and creative designs. The purpose of this project is to display artistic work in an organized and visually appealing way.

---

## 🚀 Features

- 🖼️ Gallery of handmade crafts and artwork  
- 📂 Category-based organization of items  
- 🔍 Simple and user-friendly navigation  
- 📱 Responsive design for all devices  
- ✨ Clean and modern UI

---

## 🧑‍💻 Tech Stack

- HTML5  
- CSS3  
- JavaScript

---

## 🎭 Gallery Categories

- Handmade Decoration Pieces  
- Wall Art & Paintings  
- Pottery & Clay Crafts  
- Traditional Crafts  
- Modern Art Designs  

---

## 🎯 Purpose

This project is created to:
- Showcase creative handmade crafts  
- Practice frontend development skills  
- Build a visually attractive gallery website  

---

## 📌 How to Run

1. Clone the repository  
2. Open `index.html` in your browser  
3. Browse the craft gallery
